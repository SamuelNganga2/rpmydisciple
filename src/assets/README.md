# Assets Directory

This directory contains static assets for the disciple learning website.

## Audio Files

Due to GitHub's file size limitations, large audio files have been excluded from this repository. 

To add audio files:
1. Create an `audio` subdirectory in this folder
2. Add your audio files there
3. Update the audio file references in the components

## Images

Add your image files (logos, icons, etc.) to this directory and reference them in your components.
